I chose Progressive Web Applications as my research topic because it sits at an interesting intersection of web development and security. PWAs rely on service workers that run with elevated browser privileges, which introduces attack surfaces that don't exist in traditional web apps. Understanding how PWAs work from a development standpoint informs how to assess and secure them.

About This Project

This website was built for IT 3203 Web Development at Kennesaw State University. It is completed in three milestones across the semester. This submission covers Milestone 2, which adds the JavaScript quiz application on top of the Milestone 1 site structure.

The site is built from scratch using hand-written HTML, CSS, and JavaScript. No frameworks, templates, or code generators were used. The CSS uses CSS Grid for page layouts, flexbox for navigation, and includes three selector types: element selectors, class selectors, and ID selectors. The quiz logic lives in its own JavaScript file and handles grading, colored feedback, and a full reset.
